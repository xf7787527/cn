# 地域及可用区
京东云机房分布在全球多个位置，这些位置成为地域。每个地域（region）都是一个独立的地理区域，每个地域都是完全独立的。  

京东云支持您在不同地域部署云业务，同时为了避免单地域部署可能带来的单点风险，建议在部署方案设计阶段考虑多地域多可用区部署。Kubernetes集群创建后，不支持更换地域或更换可用区。

可用区（Availability Zone）是电力及网络之间互相独立的物理区域，相同可用区内的集群之间较之同地域不同可用区内实例之间的网络延时更小。同地域内不同可用区之间提供内网互通环境，可用区之间可做到故障隔离。

* 若您的业务要求有较高容灾能力，建议将Kubernetes集群部署在同一地域不同可用区内。  
* 若您的业务要求有较低网络时延，建议将Kubernetes集群部署在同一可用区内。
