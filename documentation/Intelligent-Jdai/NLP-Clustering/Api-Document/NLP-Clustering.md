# 聚类

## 一、接口描述

### 1. 功能描述

对输入的文本数据进行聚类。

### 2. 接口使用

进入京东智联云控制台-账号管理-Access Key管理，创建并获取Access Key。整体流程详见 [调用方法](https://docs.jdcloud.com/cn/common-declaration/api/call-methods) 。

## 二、请求说明

### 1. 接口地址 ：

```
https://aiapi.jdcloud.com/jdai/corpus_aggregation
```

### 2. 请求方式：

```
 POST
```
