## **使用限制**

|         **类型**      |   **限制**           | **备注**           |
| :------------ | ----------------- | ---------------- |
|   同一个账号下可自定义创建的资源组数量   | 100 |    |
|   同一个账号下可最多拥有的资源组数量   | 101 |	101，1个默认资源组及100个自定义资源组  |
|   一个资源归属多少个资源组   | 1  |  必须归属且仅能归属一个资源组  |
|   资源组名称长度、字符限制   |   |  最多输入100个字符，仅支持汉字，字母，数字和连字符"-", "_"。 不可使用“jrn:”和“jdc-createdby”前缀  |
